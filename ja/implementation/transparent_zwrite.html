MToon を Unlit 的に設定する

Emission を使うのがこつです。

  • _MainTex: 元の _MainTex

  • _Color: (0, 0, 0, 1)

  • _ShadeTex: null

  • _ShadeColor: (0, 0, 0, 1)

  • _EmissiveMap: 元の _MainTex

  • _Emission: (1, 1, 1, 1)
